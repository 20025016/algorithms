# Lists

## Description
A list is a ordered data type which elements are stored in a specific order which makes it easier and faster at retrieving elements. A list allows for data repetition will means more than one of the same item can appear within a list.

## Code
```python
numbers = [9, 7, 7, 10, 11]

total = 0

for x in numbers:
    total = total + x

print(total)
```
